# Revenue Workflows Playbook

Use this to launch revenue quickly using `era-agents-ops`.

## 1) Build a revenue plan

```bash
echo "Create revenue from workflows, apps, ecommerce, and websites for a lean startup budget" | fabric -p era_revenue_builder
```

## 2) Run agency-first execution

```bash
echo "Build a productized AI workflow agency offer for UK SMEs" | fabric -p era_agency_triage
```

## 3) Add spec-first build for platform products

```bash
echo "Build a secure ecommerce automation platform with approval gates" | fabric -p era_sdd_triage
```

## 4) Recommended first 3 offers

1. **Workflow Ops Sprint** (£750–£2,500 setup + monthly retainer)
2. **Lead-Gen Website + Automation** (£1,000–£4,000/project)
3. **Niche Micro-SaaS MVP** (£29–£199/month pricing test)

## 5) Guardrails

- Do not trade real funds until paper-trading validation is complete.
- Use explicit compliance/security review for payments, data handling, and auth.
- Keep pricing tests short and evidence-driven.
